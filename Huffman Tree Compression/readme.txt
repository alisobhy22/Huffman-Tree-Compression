1. Copy paste the file that you wish to compress at the same folder of the .exe
2. File must hold the name "input.txt", otherwise program won't look
3. run the exe and follow instuctions for compression
4. If you wish to decompress, compressed file must hold the name "compressed.txt" otherwise it won't work
5. compressed file must be in the same folder as .exe folder
6. Run .exe file and follow instructions for decompression, decompressed file will be found in the same file under the name "decompressed.txt"
